# Guatapets

## Project Overview

This is a web application about animals in Guatap√©, Colombia, a place where you can post lost animals, found animals, and adoptable animals.

### Infrastructure

- Astro for Frontend and Server
- Bun as Runtime
- Vercel for Deployment
- Turso for Database
- Drizzle for ORM

## Scripts

- `bun run dev` for development
- `bun run build` for production
- `bun run preview` for preview

## Coding Style

- Use TypeScript for all code.
- Write Simple test
- Follow the best practices of Astro and Bun
- Use semantic HTML
